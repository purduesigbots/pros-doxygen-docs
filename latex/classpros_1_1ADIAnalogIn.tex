{\footnotesize\ttfamily std\+::int32\+\_\+t get\+\_\+value}



Reads an analog input channel and returns the 12-\/bit value. 

The value returned is undefined if the analog pin has been switched to a different mode. The meaning of the returned value varies depending on the sensor attached.

Inherited from \mbox{\hyperlink{group__cpp-adi_ga60987c8e4946650cf9aa40f8e8345f01}{A\+D\+I\+Port\+::get\+\_\+value}}.

This function uses the following values of errno when an error state is reached\+: E\+A\+D\+D\+R\+I\+N\+U\+SE -\/ The port is not configured as an analog input (e.\+g. the port has been reconfigured)

\begin{DoxyReturn}{Returns}
The analog sensor value, where a value of 0 reflects an input voltage of nearly 0 V and a value of 4095 reflects an input voltage of nearly 5 V
\end{DoxyReturn}
