\doxysubsection{Detailed Description}
Legacy L\+CD Emulator

This file defines a high-\/level A\+PI for emulating the three-\/button, U\+A\+R\+T-\/based V\+EX L\+CD, containing a set of functions that facilitate the use of a software-\/ emulated version of the classic V\+EX L\+CD module.

Visit \href{https://pros.cs.purdue.edu/v5/tutorials/topical/llemu.html}{\texttt{ https\+://pros.\+cs.\+purdue.\+edu/v5/tutorials/topical/llemu.\+html}} to learn more.
