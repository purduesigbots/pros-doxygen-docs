Turns off the Legacy L\+CD Emulator. 

Calling this function will clear the entire display, and you will not be able to call any further L\+L\+E\+MU functions until another call to lcd\+\_\+initialize.

This function uses the following values of errno when an error state is reached\+: E\+N\+X\+IO -\/ The L\+CD has not been initialized. Call \mbox{\hyperlink{group__c-motors_ga36a50f90e2186a37abce6cf36d5df6a3}{lcd\+\_\+initialize()}} first.
