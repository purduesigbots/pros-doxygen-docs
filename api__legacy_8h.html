PROS 2 Legacy API header

Contains declarations for functions that are name-compatible with the PROS 2 API. Some functions from the PROS 2 API are not useful or cannot be implemented in PROS 3, but most common functions are available.

This file should not be modified by users, since it gets replaced whenever a kernel upgrade occurs.
